# Oasis Infobyte Data Science Internship Projects

This repository contains the projects completed during the Oasis Infobyte Data Science Internship.

## Projects

### Task 1: Iris Flower Classification

* Classification of Iris flowers into Setosa, Versicolor, and Virginica species using Machine Learning.
* Techniques Used:

  * Data Preprocessing
  * Exploratory Data Analysis (EDA)
  * Classification Models
  * Data Visualization

### Task 2: Email Spam Detection

* Built a Machine Learning model to classify messages as Spam or Ham (Not Spam).
* Techniques Used:

  * Data Cleaning
  * Missing Value Handling
  * Duplicate Removal
  * Exploratory Data Analysis (EDA)
  * Feature Engineering
  * TF-IDF Vectorization
  * Multinomial Naive Bayes Classification

## Technologies Used
