# Oasis Infobyte Data Science Internship Projects

Welcome to my Data Science Internship Project Repository. This repository contains Machine Learning projects completed as part of the Oasis Infobyte Data Science Internship Program. Each project demonstrates different stages of the Machine Learning pipeline, including data preprocessing, exploratory data analysis, model development, evaluation, and prediction.

---

## Projects Included

### 🌸 Task 1: Iris Flower Classification

#### Project Overview

The Iris Flower Classification project focuses on classifying iris flowers into three species: Setosa, Versicolor, and Virginica based on their sepal and petal measurements.

#### Key Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection
* Classification model training
* Model evaluation and prediction

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

#### Machine Learning Technique

* Classification

### 📧 Task 4: Email Spam Detection

#### Project Overview

This project aims to identify whether a message is Spam or Ham (Not Spam) using Natural Language Processing and Machine Learning techniques. Spam detection plays an important role in filtering unwanted and potentially harmful messages.

#### Key Features

* Data cleaning and preprocessing
* Missing value handling
* Duplicate record removal
* Exploratory Data Analysis (EDA)
* TF-IDF Vectorization
* Spam message classification
* Model evaluation using confusion matrix and accuracy score

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

#### Machine Learning Technique

* Multinomial Naive Bayes
* Text Classification

### 📈 Task 5: Sales Prediction using Python

#### Project Overview

Sales Prediction helps businesses estimate future sales based on advertising expenditures across different media channels such as TV, Radio, and Newspaper. This project analyzes advertising data and predicts product sales using Machine Learning.

#### Key Features

* Data preprocessing and cleaning
* Missing value and duplicate handling
* Correlation analysis
* Exploratory Data Analysis (EDA)
* Sales forecasting
* Model evaluation using regression metrics
* Future sales prediction

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

#### Machine Learning Technique

* Linear Regression
