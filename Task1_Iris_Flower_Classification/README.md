# Iris Flower Classification

## Objective
To classify iris flowers into different species using Machine Learning.

## Dataset
Iris Flower Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Algorithm
K-Nearest Neighbors (KNN)

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Output
- Iris-setosa
- Iris-versicolor
- Iris-virginica
